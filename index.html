My Recent Works

Here's a showcase of my recent projects and portfolio work - each one representing unique solutions and successful collaborations with amazing clients.

Expense Smart

ExpenseSmart is a submodule of the NextGN app designed to manage employee expense invoices within companies registered on the platform under APSmart. It helps organizations track, submit, and review employee expenses in a structured and efficient way, improving transparency and simplifying expense management workflows.

Song Note

SongNote is a note-taking app designed for singers and songwriters, allowing them to capture lyrics and audio recordings together in one organized place. It helps creatives quickly record ideas, structure songs, and keep both written notes and voice recordings neatly linked for an efficient songwriting workflow.

EasyGoats

EasyGoats.com is an informational website about domestic goats, covering breeds, care, and interesting facts. It presents engaging content and helpful details for animal lovers, farmers, and anyone curious about goats.

Bizzle

Bizzle is a mobile game designed for kids, featuring engaging children’s songs and fun puzzle activities built around colorful cartoon illustrations. The game focuses on learning through play, offering an interactive and visually appealing experience that keeps young users entertained while supporting creativity and early cognitive development.

BawsHuman – Web Platform

BawsHuman is a health and fitness platform connecting athletes and trainers. I contributed to the design and development of core features using React, Django, and Python.

Bawshuman Mobile App

Bawshuman is a fitness and wellness platform connecting trainers and clients. As a mobile app developer, I contributed to building trainer–client relationship features using modern mobile technologies.

Awesome.Bible

A clean, intuitive Bible reading experience. I built the core reading features including chapter/book navigation, clean UI flows, and optimized reading contexts using React Native and Node.js.

ZippyApp

Full-stack web development across frontend, backend, and database layers using Django, Python, React, MySQL, and TypeScript. Built and maintained key features while keeping the system running smoothly.

AwesomeBible Web

A modern online Bible reading platform built with Django and React. I developed both the backend APIs and React interface for smooth chapter/verse navigation and an improved reading experience.

Misseso

A property marketplace platform for renting and selling real estate. I built backend APIs, managed database structure, developed the web interface, and led the team coordinating with mobile developers.

Marathon

A platform for Marathon Liquors' sales teams and distributors. Built with C# and SQL Server, I developed the backend, APIs, database structure, and web components while coordinating with mobile developers.

AxionGebeya

A stock exchange platform in development for Ethiopia. I worked on the backend, API, and web platform — implementing business logic, data models, and building the web interface for trading operations.

Dambi Dollo University ERP

An ERP system for university management, student registration, courses, grading, and admin workflows. I built backend, APIs, and web platform components, integrating modules into a cohesive product.

Viptaxi

A ride-hailing platform for VIP clients with luxurious cars. I built the backend, APIs, and web interfaces for ride management, coordinating with the mobile team for seamless integration.

Contrarian

A book-sharing platform for MBA students built with Next.js. I implemented frontend features for listing and browsing books, and integrated backend services for data fetching and user management.

Knowledge Management Platform

An African Union knowledge-management platform for the Peace and Security Department. I built backend, APIs, and web interface for managing documents, workflows, and user access.

Hacktoolkit Website

A community-driven platform providing libraries and tools for hackathons and rapid prototyping. I contributed to the website and developer-friendly libraries that simplify common tasks.
